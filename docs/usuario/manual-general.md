# Manual de Usuario - SGCI-IDI

## Introducción

SGCI-IDI (Sistema de Gestión de Compras e Inventario) es una aplicación web diseñada para gestionar el inventario, las compras y las solicitudes de una organización.

## Acceso al Sistema

1. Abra su navegador web
2. Ingrese la URL del sistema
3. Ingrese sus credenciales:
   - **Usuario**: Su correo electrónico
   - **Contraseña**: La proporcionada por el administrador
4. Haga clic en "Iniciar Sesión"

## Roles del Sistema

| Rol | Descripción |
|-----|-------------|
| Superadmin | Acceso total al sistema |
| Supervisor | Gestión con reportes avanzados |
| Logistica | Gestión de inventario y compras |
| Solicitante | Solo puede crear solicitudes |

## Interfaz Principal

### Elementos de la Interfaz
- **Barra lateral izquierda**: Menú de navegación
- **Cabecera**: Nombre del usuario, notificaciones, cerrar sesión

## Glosario de Términos

| Término | Definición |
|---------|------------|
| Stock | Cantidad de producto disponible |
| SKU | Código identificador único del producto |
| RFQ | Solicitud de cotización a proveedores |
| OC | Orden de Compra |
| Kit | Conjunto de productos combinados |

## Conceptos Importantes

### Estados de Productos
- **Activo**: Disponible para usar y solicitar
- **Inactivo**: No disponible temporalmente

### Estados de Solicitudes
- **Pendiente**: Esperando aprobación
- **Aprobada**: Aprobada y lista para entregar
- **Rechazada**: No aprobada
- **Completada**: Entregada al solicitante

### Estados de Órdenes de Compra
- **Borrador**: Sin enviar
- **Emitida**: Enviada al proveedor
- **Completada**: Recepcionada
- **Anulada**: Cancelada
